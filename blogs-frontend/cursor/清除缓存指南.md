# 清除缓存完整指南

## 问题确认
- ✅ 服务器端已更新（curl 测试找到中文内容）
- ✅ 服务正常运行
- ❌ 浏览器显示旧版本

## 清除浏览器缓存

### 方法一：强制刷新（最快）
- **Windows/Linux**: `Ctrl + Shift + R` 或 `Ctrl + F5`
- **Mac**: `Cmd + Shift + R`

### 方法二：清除浏览器缓存
1. 按 `Ctrl + Shift + Delete`（Windows/Linux）或 `Cmd + Shift + Delete`（Mac）
2. 选择"缓存的图片和文件"
3. 时间范围选择"全部时间"
4. 点击"清除数据"

### 方法三：使用无痕模式
- **Windows/Linux**: `Ctrl + Shift + N`
- **Mac**: `Cmd + Shift + N`
- 在无痕窗口中访问 `http://119.91.150.19`

### 方法四：开发者工具清除
1. 按 `F12` 打开开发者工具
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

## 检查 Nginx 缓存配置

### 宝塔面板 Nginx 配置位置

```bash
# 检查 Nginx 配置目录
ls -la /www/server/nginx/conf/

# 检查是否有缓存配置
grep -r "proxy_cache" /www/server/nginx/conf/

# 检查站点配置
ls -la /www/server/panel/vhost/nginx/
```

### 清除 Nginx 缓存（如果有）

```bash
# 清除 Nginx 缓存目录（如果存在）
rm -rf /www/server/nginx/cache/*
rm -rf /tmp/nginx_cache/*

# 重启 Nginx
nginx -s reload
# 或
/etc/init.d/nginx reload
```

## 检查是否有 CDN/代理

如果使用了 CDN 或反向代理，可能需要：
1. 清除 CDN 缓存
2. 等待 CDN 缓存过期
3. 或强制刷新 CDN 缓存
